# Sonjas Roblox-spel 🎮

Ett litet 3D-spel (Roblox-stil) som körs direkt i webbläsaren. Byggt med [three.js](https://threejs.org/) (laddas från CDN) — **allt ligger i en enda fil: `index.html`**, inga installationer behövs.

## Spela
Dubbelklicka på `index.html` så öppnas spelet i webbläsaren.

### Kontroller
- **Piltangenter / WASD** – gå
- **Mellanslag** – hoppa (högt!)
- **Dra med musen** – titta runt (snurra kameran)
- **Scrolla** – (avatar-läget) zooma
- **📋 Koder** (uppe till höger) – lista med riktiga koder du kan kopiera
- Knapparna nere till vänster + **HOPP** funkar på pekskärm/telefon

### Att göra i spelet
- Samla alla **8 stjärnor** ⭐
- Klara **obbyn** (hoppbanan med plattformar) – upp till målflaggan 🏁
- Hoppa upp på de färgade klossarna
- Spring runt med din **avatar** + husdjuren (🐹 hamster + 🐱 katt)
- Säg hej till de andra spelarna (Max, Liv, Noah, Ella)

## Koderna 🔑
På väggarna (och i 📋 Koder-listan) står **riktiga koder** för Roblox-spelen
**99 Nights in the Forest** och **Dress to Impress**.

> ⚠️ Koder går ut med tiden! De här kollades **2026-06-22** mot flera kodsajter
> (Pocket Tactics, Pro Game Guides, Beebom, PCGamesN, Pocket Gamer).
> Skriv dem exakt som de står: STORA bokstäver, och `0` betyder noll (inte bokstaven O).

## Publicera senare (Render)
Det här är en statisk sida (bara `index.html`). När ni vill lägga ut den på nätet:
1. Skapa ett GitHub-repo och lägg upp den här mappen.
2. På [Render](https://render.com): **New → Static Site**, koppla repot.
3. Build command: *(tomt)* · Publish directory: `.` (rotmappen).

Då får spelet en länk som vem som helst kan öppna. 🌍
